# Save Restricted Content Bot

A Telegram bot to save restricted content from channels and groups.

## Features
- Save content from private/restricted channels.
- Batch processing for multiple messages.
- Premium membership management.
- Automated renewal reminders.
- Custom branding via environment variables.

## Deployment
You can deploy this bot on any VPS or platform that supports Python.

### Environment Variables
- `API_ID`: Your Telegram API ID.
- `API_HASH`: Your Telegram API Hash.
- `BOT_TOKEN`: Your Telegram Bot Token.
- `MONGO_DB`: Your MongoDB connection string.
- `OWNER_ID`: Your Telegram User ID.
- `BOT_NAME`: Name of your bot.
- `OWNER_USERNAME`: Your Telegram username.
- `JOIN_LINK`: Link to your Telegram channel.
- `ADMIN_CONTACT`: Link to contact admin.
- `START_PIC`: URL for the start picture.
